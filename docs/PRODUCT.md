# PRODUCT: Happiness Arcade

## One-line Pitch
A joyful micro-app that helps people feel better daily through mood tracking, gratitude, and tiny positive actions.

## Target User
- Students and working professionals
- People who want a simple well-being routine without long journaling

## Core Loops
1. Open app once a day.
2. Pick mood (1–5).
3. Write one gratitude line.
4. Complete a tiny quest.
5. See streak and 7-day progress.

## v0 Features
- Daily check-in card
  - Mood slider/buttons (1–5)
  - Gratitude input (single line)
  - Tiny quest complete checkbox
- Save/check-in history in localStorage
- 7-day timeline list
- Streak counter

## Non-Goals (v0)
- No authentication
- No backend API
- No social features

## Future Ideas
- Auto-generated weekly “Feel-Good Replay”
- Badges and challenge packs
- Optional account sync
